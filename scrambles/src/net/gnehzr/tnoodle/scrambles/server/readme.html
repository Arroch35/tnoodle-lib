ScrambleServer

A many purpose http server for twisty puzzles.

Instructions

Simply point your browser to /scrambler/ to get a list of available puzzles in JSON format.
Choose one of these to get a scramble: /scrambler/3x3x3


Examples

/scrambler/3x3x3?seed=cubing&count=12 /scrambler/3x3x3.json /scrambler/3x3x3.OH Round 1.pdf?count=5



Features

  • All pages serving up JSON also support JSONP via a callback parameter.

/scrambler/[puzzlename[[.title].ext]?seed=SEED&count=COUNT&callback=CALLBACK&scheme=SCHEME

VariableDescriptionValue RangeDefaultComment
.extscript extension .txt | .json | .pdf .txt The extension will determine if the scrambles are generated as plaintext, json, or in a pdf with images.
.titletitleAny string This parameter only applies if .ext is .pdf, in which case the title will appear on top of the generated pdf.
seedseedAny stringSome (hopefully) secure random number. This is extremely useful for racing people, as two people can now simply agree upon a seed and then independently go to the ScrambleServer to get THE SAME scrambles!
countscramble countinteger1 The number of desired scrambles.
offsethashed scrambles offsetinteger0 An offset into the list of scrambles from which to start generating scrambles (this is only really useful if using seeded scrambles).
callbackjsonp callbackThe name of a javascript function This parameter only applies if the extension is .json, if so, the resulting json will be padded accordingly.
schemecolor schemeA comma separated list of html colors or one of [ybrwgop0] for short. The colors are ordered by the faces in alphabetical order. This only applies if .ext is .pdf.

/viewer/puzzlename.ext?scramble=SCRAMBLE&width=WIDTH&height=HEIGHT&callback=CALLBACK

VariableDescriptionValue RangeDefaultComment
.extscript extension .png | .json If .png, then the scramble will be drawn as a png file. If .json, then boundaries of the faces will be returned in json format.
widthwidthinteger0 This is the maximum allowed width for the resulting image. It will not screw up the image's aspect ratio. If 0, it will attempt to use the preferred width.
heightheightinteger0 This is the maximum allowed height for the resulting image. It will not screw up the image's aspect ratio. If 0, it will attempt to use the preferred height.
callbackjsonp callbackThe name of a javascript function This parameter only applies if the extension is .json. If so, the resulting json will be padded accordingly.
schemecolor scheme See above
